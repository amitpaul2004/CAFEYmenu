## вңЁ About BrewVerse Elite

**BrewVerse Elite** is a modern, high-performance **Real-Time Restaurant Management System** created for smooth communication between:

вң… Customers  
вң… Kitchen Staff  
вң… Restaurant Owners (Admin)

рҹҡҖ Each terminal can run on different devices but stays synced in real-time using **WebSocket / Socket.io**.

## рҹӣ пёҸ Tech Stack (Updated)

вң… This project **does NOT use MERN Stack**.

- **Frontend**: HTML5, CSS3 (Advanced Glassmorphism), JavaScript (ES6+)
- **Backend**: Node.js + Express.js
- **Real-Time Engine**: Socket.io (WebSockets)
- **Database**: MongoDB Atlas *(if used)*
- **Deployment**:
  - **Vercel / Netlify**: Frontend terminals
  - **Render / Railway**: Backend (WebSocket hub)
